## Pan, zoom, and select like your favorite design tools with the v11.4.0 minor release üí´

In the [latest minor release](https://github.com/wbkd/react-flow/releases/tag/11.4.0), you can finally configure the controls so that they behave like Figma, Miro or Sketch. This includes click-dragging to make a selection, [pan by holding the spacebar, and scrolling to zoom](https://reactflow.dev/docs/guides/panning-and-zooming/). You can also configure if a node is selected if it is fully inside/partially inside the selection area.

**[Resize and Rotate](https://reactflow.dev/docs/examples/nodes/resize-rotate/):** This was a Pro example, but we decided to make the source code available to everyone in our public docs to avoid trouble people were having with third-party resizing libraries.

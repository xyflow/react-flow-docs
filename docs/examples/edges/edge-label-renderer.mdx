If you want to escape SVG world within an edge, you can use the [`<EdgeLabelRenderer />`](/docs/api/edges/edge-label-renderer/). It's a portal component that lets you render edge lables as divs on top of the edges. If you want to add mouse interactions you need to set `pointer-events: all` on the label.

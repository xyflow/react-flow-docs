This is an example of a [custom edge](/docs/api/edges/custom-edges/) implementation with a button to remove the underlying edge. In order to leave SVG world we are using the [`foreignObject` element](https://developer.mozilla.org/en-US/docs/Web/SVG/Element/foreignObject) inside the custom edge to be able to use an HTML `button`.

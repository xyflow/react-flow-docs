Smooth transitions from one viewport to another are built-in. You can pass a `duration` option parameter to define the animation duration for every function that alters the viewport ([`zoomIn`, `zoomOut`, `zoomTo`, `setViewport`, `fitView`, `setCenter`, `fitBounds`](/docs/api/react-flow-instance/#viewport)).

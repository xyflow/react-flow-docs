Custom nodes need to have at least one [`Handle` component](/docs/api/nodes/handle/) to be connectable. You can pass a validation function [`isValidConnection`](/docs/api/react-flow-props/#isvalidconnection-1) to the ReactFlow component in order to check if a new connection is valid and should be added.
